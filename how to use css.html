How to use the CSS [Resource Swapper]

How to use the CSS:
[Keep in mind the steps are different depends on what you play on]
  • For Client [Given steps are for official client, for other clients the steps are similar]
    1. Go to This PC > Documents > KrunkerResourceSwapper
    2. Create a folder, and name it css [all lowercase letters]
    3. Take the css file, which should be named main_custom, and place it inside the css folder
    4. Open the client, go to your settings and make sure the Resource Swapper is enabled, and reload your client
    5. You are then good to go!

  • For Browsers [You need a program that can operate on .zip files, like WinZip]
    1. Download the zip file from Tehchy's Resource Swapper, extract it, and open the extracted folder
    2. Do not delete any of the files that are already inside there
    3. Create a folder, and name it css [all lowercase letters]
    4. Take the css file, which should be named main_custom, and place it inside the css folder
    5. Open your browser, go to the extensions page, and turn on developer mode [In Brave its at the top right]
    6. Drag the KrunkerResourceSwapperMaster folder into the extensions page.
    7. Open the game and enjoy!
